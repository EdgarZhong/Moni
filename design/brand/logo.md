# Logo

## Header Logo

- Header 使用完整 `Moni` 字标。
- Header 中的 `M` 不做猫耳图标化处理。
- 字标右上区域固定保留品牌三色装饰簇。
- 三色装饰必须聚在 `M` 与 `oni` 的交界上方，不能漂到最右侧。

## 底部中央按钮

- 底部中央按钮承担品牌入口、首页入口、AI 状态与 AI 控制四重职责。
- 中央按钮允许使用更强猫耳感的 `M` 变体。
- 中央按钮中的 `M` 与三色装饰应整体更大、更有存在感。

## App Icon

- App Icon 与底部中央按钮共用同一套核心图形语言。
- 初版允许在中央按钮 SVG 基础上做适配输出 PNG。

## 约束

- Header Logo 与中央按钮是同源不同职，不得直接互换。
- 若修改字标构图、装饰位置或 App Icon 基线，必须新增 DDR。
