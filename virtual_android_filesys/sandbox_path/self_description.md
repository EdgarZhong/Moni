我是西北工业大学学生，和女朋友一起生活，正餐只统计两个人一起用餐的情况。
具体来说，比如说一个正餐时间段内，如果消费只有单笔小额，比如说十几块钱、20块钱以内的情况，那基本上是我一个人在吃饭。如果说有一笔消费比较高的，如30多、40多，那可能是我一次性支付了两个人的饭费，如果都是十几，但是有两笔，那么可能是我分两次支付了两个人的饭费。你必须判断到底是我一个人在用餐还是两个人在用餐，正餐只记录两个人的，一个人用餐的这个餐饮消费算在其它里。
另外，凡是杨国福麻辣烫的，肯定都是双人餐。价格不高是因为我直接用银行卡支付了美团团购券，剩下的小额只是补差价。给杨国福的消费，全部计算在正餐中。
餐饮消费可能出现一个小时段多笔，如果加一起超过75元算大餐，相应时段这些，都应该归入大餐当中，不要再往正餐当中放。
